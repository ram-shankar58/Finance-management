## SETUP
create config folder and add config.env file in it and all all env variables there.

`MONGO_URL` : Your MongoDB Connection String

`PORT`: PORT number

## Run-up

```bash
  cd backend
```
```bash
  npm install
```

```bash
  npm start
```

```bash
  npm run dev
```
